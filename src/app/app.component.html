Song Sync

What is this all about??

Sync song - As name suggests, play a song on multiple devices in sync. Web app starts playing song on multiple devices at specified time. Now default song loads. In future, you can upload your own song and if someone else uses your session ID then your song will be loaded on their device and played at the time you have specified.

How to use?
-> Session: A session is created by default, when you visit this web app.
-> Set Time: Set the time (in seconds) after which song should start.
-> Share Session ID: Share session ID with friends near by. They have to enter the session ID in "Use Existing Session" Tab.
-> And see the magic happen!

To Create new session

Click on "Create Session" Tab => Enter the seconds after which song should start. => click "Set Time".

To Use Existing session

Click on "Use Existing Session" Tab => Enter the Session ID shared by your friend. => click "Set Time".

Song will play after the specifed amount of time. Check the count down, to see when song starts.
